== Description ==
RomethemeForm For Elementor Plugin is an Form Builder for Elementor, and Widget Ready to use.

New :
• Number Widget
• Telephone Widget

Features
• Form Builder
• Entries View
• Send Email Notification and Email Confirmation
• Can Create new form in Elementor Builder

Form Widget
• Text Field
• Email Field
• Text Area Field
• Date Field
• Time Field
• Submit Button
• Select Widget
• Radio Button Widget
• Checkbox Widget
